# GifTastic

Used the GIPHY API to make a dynamic web page that populates with gifs of your choice. Called the GIPHY API and used JavaScript and jQuery to change the HTML of site.

Used:
1) HTML
2) CSS
3) Javascript
4) GIPHY API
5) jQuery
6) ajax
